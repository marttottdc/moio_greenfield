# resources

## Responsibility

Shared API resources and utilities for cross-application functionality.

## What it Owns

- Contact search endpoint
- WhatsApp template management
- Agent tools and tools registry

## What it Does NOT Do

- Does not manage contacts directly (reads from crm)
- Does not manage tenant configuration (reads from portal)
- Does not send messages directly (delegates to chatbot)
